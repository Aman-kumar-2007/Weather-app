# 🌦️ Weather App (API Based)

## 📌 Overview

This Weather App is a simple and responsive web application that fetches real-time weather data using a public weather API. Users can search for any city and get current weather details such as temperature, humidity, wind speed, and weather conditions.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* 💧 Humidity and wind speed info
* 🌤️ Weather condition icons (clear, clouds, rain, etc.)
* 📱 Fully responsive design
* ⚡ Fast API-based data fetching

---

## 🛠️ Tech Stack

* **HTML** – Structure
* **CSS** – Styling & responsiveness
* **JavaScript** – Logic & API handling
* **Weather API** – Fetch live weather data

---

## 🔑 API Used

You can use any weather API like:

* OpenWeatherMap API

👉 Example API Endpoint:

```
https://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}&units=metric
```

---


## ⚙️ How It Works

1. User enters a city name
2. JavaScript sends a request to the weather API
3. API returns weather data in JSON format
4. Data is parsed and displayed on the UI
